# Desafio Banco — Como rodar

Resumo rápido para executar a aplicação Python que está no arquivo `app.py`.

Requisitos
- Python 3+ instalado no Windows.

Passos
1. Abra o PowerShell ou Prompt de Comando.
2. Vá para a pasta do projeto:
3. Execute a aplicação:
   ```
   python app.py
   ```

Uso básico
- O programa mostra um menu com opções:
  - [u] Cadastrar Usuario (cliente)
  - [c] Cadastrar Conta
  - [d] Depositar
  - [s] Sacar
  - [e] Extrato
  - [q] Sair
- Siga as instruções na tela para informar CPF, número da conta e valores.

Observações
- Não há dependências externas; apenas Python.
- Se surgir erro ao converter entradas numéricas, digite valores válidos (somente números).
- A conta é identificada por número com 4 dígitos (ex.: `0001`).
